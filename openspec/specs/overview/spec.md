## Purpose

OpenSpec is a CLI tool designed to analyze codebases, detect architectural patterns, and generate
OpenSpec documentation. It ensures alignment between code and architectural decisions by tracking
drift, verifying specifications, and maintaining up-to-date documentation of system behavior and
requirements.
